# Decision Tree Classifier - Bank Marketing Dataset

## Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

## Dataset
Bank Marketing Dataset from UCI Machine Learning Repository.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Prediction
8. Model Evaluation
9. Decision Tree Visualization
10. Feature Importance Analysis

## Model Used
Decision Tree Classifier

## Accuracy Achieved
91.47%

## Outcome
Successfully predicted customer purchase behavior using machine learning techniques.
